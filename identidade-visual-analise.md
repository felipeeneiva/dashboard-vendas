# Análise da Identidade Visual - Mundo Pró Viagens

## Site Oficial: www.mundoproviagens.com.br

### Cores Principais
1. **Azul claro vibrante** (#a0d3f0 ou similar) - usado no logo "MUNDO PRÓ" e títulos grandes
2. **Laranja vibrante** (#fd4f0d ou #ff5722) - usado em:
   - Botão de navegação arredondado no topo
   - Texto manuscrito "Tão legal que quase nem parece agência"
   - Seção "Cuidamos de tudo" (fundo laranja forte)
3. **Fundo escuro** (#2d2d2d ou #333) - predominante no hero
4. **Branco** - textos principais

### Tipografia
1. **Logo**: Fonte bold, maiúscula, moderna (MUNDO PRÓ)
2. **Títulos grandes**: Sans-serif bold, branco
3. **Texto manuscrito/casual**: Fonte script/handwriting para dar personalidade ("Tão legal que quase nem parece agência")
4. **Corpo de texto**: Sans-serif limpa

### Elementos Visuais Característicos
1. **Botão de navegação laranja arredondado** no topo (muito característico)
2. **Fundo escuro** (não bege claro como aplicamos)
3. **Contraste forte**: Azul claro + Laranja sobre fundo escuro
4. **Estilo moderno e jovem**: menos corporativo, mais descontraído
5. **Texto manuscrito laranja** para dar personalidade
6. **Logo azul claro** sempre visível

### Diferenças do Dashboard Atual
❌ Fundo bege claro → Deveria ser **fundo escuro (#2d2d2d)**
❌ Cards com bordas coloridas → Deveria ter **mais contraste, fundo escuro**
❌ Logo não aparece → **Logo azul claro deve estar sempre visível**
❌ Estilo muito corporativo → Deveria ser **mais jovem e descontraído**
❌ Falta elemento manuscrito → Adicionar **textos em fonte script laranja**

### Ações Necessárias
1. Mudar fundo de bege claro para **cinza escuro (#2d2d2d ou #1a1a1a)**
2. Logo azul claro **sempre visível** no sidebar
3. Usar **azul claro** para títulos principais (não preto)
4. Manter **laranja vibrante** para CTAs e destaques
5. Adicionar **fonte manuscrita** (Dancing Script ou Pacifico) para elementos especiais
6. Cards com **fundo escuro + bordas coloridas brilhantes**
7. Mais **contraste** e menos tons pastéis
